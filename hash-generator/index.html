Hash Generator

Generate MD5, SHA1, SHA256, SHA512 hashes from text.

About Hash Functions

Hash functions convert data of arbitrary size into a fixed-size hash value. They are used for data integrity, password storage, and digital signatures.

MD5: Fast but not secure, used for checksums
SHA-1: Deprecated for security use
SHA-256: Part of SHA-2, widely used
SHA-512: Stronger, used for high security

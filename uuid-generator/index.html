What is a UUID?

A UUID (Universally Unique Identifier) is a 128-bit identifier that is unique across all devices and time. UUIDs are commonly used in software development for:

  • Database primary keys
  • Session identifiers
  • Transaction IDs
  • Distributed systems

Format: xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx

Barcode Format Guide

Code 128

Most versatile. Supports all ASCII characters. Used in shipping labels.

Code 39

Supports A-Z, 0-9, and special characters. Common in automotive and defense.

EAN-13

International retail product code. 13 digits with check digit.

UPC-A

North American retail standard. 12 digits with check digit.
